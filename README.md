## About The Project
**text-identity** - __script, that helps you to compare two text files and find a percent difference between them__

<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bekhruzdilshod/text-identity.git
   ```

<!-- USAGE EXAMPLES -->
## Usage
1. Install script and copy paths to the two selected files you want to compare    
2. Run script via your OS terminal:   
    ```shell
    python identity.py -first 1.txt -second 2.txt
   ```
3. Get result 
    ```shell
    Identity percent between two files: 95.04%
    ```
